# cursosonlineandre
PLATAFORMA DE CURSOS ONLINE BÁSICA
-----------------------------------------------------
Autenticação


RF01

O sistema deve permitir cadastro de usuários.

RF02

O sistema deve permitir login com email e senha.

RF03

O sistema deve permitir logout do usuário.

-----------------------------------------------------

Cursos

RF04

O sistema deve listar todos os cursos disponíveis.

RF05

O sistema deve permitir visualizar detalhes de um curso.

-----------------------------------------------------

Aulas :

RF06

O sistema deve permitir assistir aulas em vídeo.

RF07

O sistema deve permitir marcar uma aula como concluída.

------------------------------------------------------

Dashboard
RF08

O sistema deve exibir os cursos iniciados pelo usuário.

------------------------------------------------------
Administração simples

RF09

O sistema deve permitir cadastrar cursos.

RF10

O sistema deve permitir inserir video de aulas em um curso.

RF11

O sistema deve permitir editar cursos e aulas.

RF12

O sistema deve permitir remover cursos e aulas.

--------------------------------------------------------
Requisitos Não Funcionais (RNF)
-

Performance

RNF01

O sistema deve carregar páginas de forma responsiva e otimizada.

--------------------------------------------------------
Segurança

RNF02

As senhas dos usuários devem ser armazenadas criptografadas.

RNF03

Rotas privadas devem exigir autenticação.


--------------------------------------------------------
Usabilidade

RNF04

O sistema deve possuir interface intuitiva e de fácil navegação.


---------------------------------------------------------
Banco de Dados

RNF05

O sistema deve utilizar Prisma ORM para manipulação do banco.

---------------------------------------------------------
Tecnologias

RNF06

A estilização deve utilizar Tailwind CSS.

RNF07

A autenticação deve utilizar Better Auth.
